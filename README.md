# ArchAi – AI Architecture Assistant

An AI-powered architecture assistant web application built with React, Vite, and a Python ML backend, deployed on Firebase.

## Tech Stack
- **Frontend:** React.js, Vite, CSS
- **Backend:** Python
- **ML:** Custom-trained model using JSONL datasets
- **3D Processing:** Blender (via Python scripting)
- **Deployment:** Firebase

## Features
- AI-powered architecture recommendations
- Custom ML pipeline with preprocessing, evaluation, and testing phases
- 3D model processing integrated via Blender Python scripting
- Full-stack deployment on Firebase

## Project Structure
- `src/` – React frontend
- `public/` – Static assets
- `backend/` – Python backend server
- `ml_pipeline/` – ML training and evaluation scripts
- `models/` – Trained model data
- `tests/` – Test suite and results
